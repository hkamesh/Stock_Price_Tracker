# Real-Time Multi Stock Price Tracker  

## Description  
This project tracks real-time stock prices from Yahoo Finance using Python and Tkinter. It supports multiple stocks and auto-refreshes every 5 seconds.

## Requirements  
- Python 3.x  
- Libraries: `yfinance`, `tkinter`, `threading`  

## How to Run  
1. Install dependencies:  
```bash
